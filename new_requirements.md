# 网页新需求记录

- 创建日期：2026-03-23
- 负责人：产品 + 前端
- 状态：讨论中

## 1. 背景与目标
- 背景：当前主页面为投资者视角，用户希望在同一页面快速切换到融资方视角演示，且不做大规模页面重构。
- 目标：新增“视角切换”按钮，点击后有反转特效；切换到融资方视角时，主页面移除/隐藏“评估通（筛子）”相关展示。

## 3. 验收标准
- [ ] AC-001：主页面可见“视角切换”按钮，点击后视角文案和按钮状态同步变化。
- [ ] AC-002：切换动作有反转动效，页面无白屏、无跳页、无控制台报错。
- [ ] AC-003：融资方视角下不展示“评估通·AI筛子”区域及筛后统计信息。
- [ ] AC-004：融资方视角下机会列表按全部机会展示，不受当前筛子影响。
- [ ] AC-005：刷新页面后保持用户上次选择的视角。

## 4. 约束与边界
- 技术约束：
  - 本次仅做前端改造，不改后端 API。
  - 仅改 Dashboard 主页面，项目会话页（Research/Workbench/Intent/Negotiation/Timeline）暂不改逻辑。
  - 保持现有模块拆分结构（`src/pages` + `public/static/js`）。
- 业务边界：
  - 本次“融资方视角”是展示视角切换，不是独立的新业务流。
  - 不新增融资方专属数据模型，不改权限体系。
